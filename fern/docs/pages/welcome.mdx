## Core capabilities

Build intelligent conversational agents with these integrated capabilities:

| Capability                   | What you can do                                                    |
| ---------------------------- | ------------------------------------------------------------------ |
| **Task Management**          | Create and manage conversational tasks with full lifecycle control |
| **Real-time Communication**  | Connect via WebSocket for bidirectional streaming communication    |
| **Message Handling**         | Submit and retrieve messages with rich context and metadata        |
| **Neuro-Symbolic Reasoning** | Leverage Apollo-1's breakthrough reasoning capabilities            |

<Tip>
New to the AUI API? Start with the [Overview](/overview) to understand the platform, then follow the [Quickstart](/quickstart) to create your first task.
</Tip>

## What you can build

- **Conversational AI Agents** - Intelligent agents that function on behalf of entities
- **Customer Support Systems** - AI-powered support with stateful reasoning
- **Task-Oriented Dialogue** - Agents that can complete complex multi-step tasks
- **Real-time Applications** - Streaming communication with instant responses
