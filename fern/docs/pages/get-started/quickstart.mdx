This guide will walk you through creating your first task and sending your first message using the AUI API.

## Prerequisites

Before you begin, make sure you have:

- An AUI API key (contact the AUI team to obtain one)
- A tool to make HTTP requests (curl, Postman, or your preferred HTTP client)

## Step 1: Install the SDK

Choose your preferred language and install the SDK:

<CodeBlocks>
  <CodeBlock title="TypeScript">
    ```bash
    npm install @aui.io/aui-client
    ```
  </CodeBlock>
  <CodeBlock title="Python">
    ```bash
    pip install aui-client
    ```
  </CodeBlock>
</CodeBlocks>

## Step 2: Initialize the Client

<CodeBlocks>
  <CodeBlock title="TypeScript">
    ```typescript
    import { ApolloClient } from '@aui.io/aui-client';

    const client = new ApolloClient({
      networkApiKey: 'YOUR_API_KEY'
    });
    ```
  </CodeBlock>
  <CodeBlock title="Python">
    ```python
    from aui import ApolloClient

    client = ApolloClient(
      network_api_key='YOUR_API_KEY'
    )
    ```
  </CodeBlock>
</CodeBlocks>

## Step 3: Create a Task

<CodeBlocks>
  <CodeBlock title="TypeScript">
    ```typescript
    const taskResponse = await client.controllerApi.createTask({
      user_id: 'user_123'
    });

    console.log('Task ID:', taskResponse.id);
    console.log('Welcome:', taskResponse.welcome_message);
    ```
  </CodeBlock>
  <CodeBlock title="Python">
    ```python
    task_response = client.controller_api.create_task(
      user_id='user_123'
    )

    print(f'Task ID: {task_response.id}')
    print(f'Welcome: {task_response.welcome_message}')
    ```
  </CodeBlock>
</CodeBlocks>

Response:

```json
{
  "id": "6914a707cf58704fbbe7b9ad",
  "user_id": "user_123",
  "title": "New Task",
  "welcome_message": "Welcome. I am a goal-oriented agent, powered by Apollo, designed to find real-time products. What are you looking for?"
}
```

## Step 4: Send a Message

Send a message and get an agent response (non-streaming):

<CodeBlocks>
  <CodeBlock title="TypeScript">
    ```typescript
    const messageResponse = await client.controllerApi.sendMessage({
      task_id: taskResponse.id,
      text: 'I need a microwave with at least 20 liters capacity',
      is_external_api: true
    });

    console.log('Agent response:', messageResponse.text);
    
    // Check for product recommendations
    if (messageResponse.cards && messageResponse.cards.length > 0) {
      messageResponse.cards.forEach(card => {
        console.log(`Product: ${card.name}`);
      });
    }
    ```
  </CodeBlock>
  <CodeBlock title="Python">
    ```python
    message_response = client.controller_api.send_message(
      task_id=task_response.id,
      text='I need a microwave with at least 20 liters capacity',
      is_external_api=True
    )

    print(f'Agent response: {message_response.text}')
    
    # Check for product recommendations
    if hasattr(message_response, 'cards') and message_response.cards:
      for card in message_response.cards:
        print(f'Product: {card.name}')
    ```
  </CodeBlock>
</CodeBlocks>

## Step 5: Retrieve Messages

Get all messages from your task:

<CodeBlocks>
  <CodeBlock title="TypeScript">
    ```typescript
    const messages = await client.controllerApi.getTaskMessages(taskResponse.id);

    console.log(`Total messages: ${messages.length}`);
    
    messages.forEach(msg => {
      console.log(`[${msg.sender.type}]: ${msg.text}`);
    });
    ```
  </CodeBlock>
  <CodeBlock title="Python">
    ```python
    messages = client.controller_api.get_task_messages(task_id=task_response.id)

    print(f'Total messages: {len(messages)}')
    
    for msg in messages:
      print(f'[{msg.sender.type}]: {msg.text}')
    ```
  </CodeBlock>
</CodeBlocks>

## Step 6: Real-time Streaming (WebSocket)

For real-time streaming communication, use the WebSocket API:

<CodeBlocks>
  <CodeBlock title="TypeScript">
    ```typescript
    const socket = await client.apolloWsSession.connect();

    socket.on('open', () => {
      console.log('âœ… Connected');
      
      socket.sendUserMessage({
        task_id: taskResponse.id,
        text: 'Show me gaming laptops under $1500'
      });
    });

    socket.on('message', (message) => {
      // Streaming updates
      if (message.channel?.eventName === 'thread-message-text-content-updated') {
        console.log('Agent:', message.data?.text);
      }
      
      // Final response with product cards
      if (message.id && message.text) {
        console.log('Complete:', message.text);
        
        if (message.cards) {
          message.cards.forEach(card => {
            console.log(`Product: ${card.name}`);
          });
        }
        
        socket.close();
      }
    });
    ```
  </CodeBlock>
  <CodeBlock title="Python">
    ```python
    from aui import ApolloClient, ApolloWsSessionClient

    client = ApolloClient(network_api_key='YOUR_API_KEY')
    apollo_ws = ApolloWsSessionClient(client_wrapper=client._client_wrapper)

    with apollo_ws.connect() as socket:
      # Send message
      socket.send_user_message({
        'task_id': task_response.id,
        'text': 'Show me gaming laptops under $1500'
      })
      
      # Iterate over messages
      for message in socket:
        # Streaming updates
        if hasattr(message, 'channel') and message.channel:
          if message.channel.event_name == 'thread-message-text-content-updated':
            print(f'Agent: {message.data.text}')
        
        # Final message
        elif hasattr(message, 'id') and hasattr(message, 'text'):
          print(f'Complete: {message.text}')
          
          if hasattr(message, 'cards') and message.cards:
            for card in message.cards:
              print(f'Product: {card.name}')
          
          break
    ```
  </CodeBlock>
</CodeBlocks>
